# 📝 Advanced To-Do List Web Application

A modern and interactive **To-Do List Manager** built using **HTML, CSS, and Vanilla JavaScript**. This project allows users to create, organize, prioritize, reorder, and complete tasks through an intuitive user interface.

---

## 🚀 Features

### ✅ Task Management

* Create new tasks with a custom task name.
* Automatically records the creation date.
* Delete tasks instantly.
* Mark tasks as completed.

### ⭐ Priority System

* Star important tasks.
* Starred tasks automatically move to the top of the list.
* Unstar tasks whenever needed.

### 🎯 Completed Tasks Section

* Completed tasks are moved to a separate **Finished Tasks** section.
* Visual strike-through effect for completed tasks.
* Tasks can be restored by unchecking them.

### 🔄 Drag & Drop Reordering

* Reorder tasks easily using drag-and-drop.
* Dedicated drag handle prevents accidental dragging.
* Smooth visual feedback while dragging.

### ⌨️ Keyboard Shortcuts

| Key    | Action                    |
| ------ | ------------------------- |
| Enter  | Create Task               |
| Escape | Close Task Creation Popup |

### 🎨 User Interface

* Clean and responsive design.
* Interactive hover tooltips.
* Modal popup for task creation.
* Blur background effect while creating tasks.
* Modern color palette and visual hierarchy.

## ⚙️ How It Works

### Creating a Task

1. Click the **Add Task** button.
2. Enter the task description.
3. Press **Create Task** or hit **Enter**.

### Starring a Task

* Click the ⭐ icon.
* The task is moved to the top of the task list.

### Completing a Task

* Check the completion checkbox.
* The task moves to the **Finished Tasks** section.

### Reordering Tasks

* Drag a task using the reorder handle.
* Drop it into the desired position.
